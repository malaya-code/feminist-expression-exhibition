## About These Citations

This exhibition represents practice-based research grounding artistic work in feminist theory and scholarly discourse. Each piece engages directly with specific feminist concepts, texts, and creators.

By linking visual art to theoretical frameworks, this project demonstrates that anyone can document and credit their creative work scholarly. Art is research. Theory informs practice. Practice extends theory.

### Supporting the Creators

Where possible, links direct to official websites and purchasing options. Please support these artists, writers, and theorists whose work made this exhibition possible.

## How to Cite Artworks from This Exhibition

### Individual Artwork (APA 7th Edition)

```
Wright, M. (2021). [Artwork title] [Medium]. Feminist Expression, In Action! exhibition. 
https://www.malayawonders.info/art/feminist-expression-exhibition
```

### Example:

